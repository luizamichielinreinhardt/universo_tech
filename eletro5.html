Arduíno: qual a linguagem de programação usada (C++, Python, ...)







Qual linguagem de programação o Arduino usa?

O Arduino é programado principalmente na linguagem C/C++, que é a base oficial usada na Arduino IDE. Essa linguagem permite controlar cada detalhe do hardware, como sensores, LEDs e motores, tornando os projetos rápidos e eficientes. Embora existam outras formas de programar o Arduino, como Python ou blocos visuais, o C/C++ continua sendo a linguagem mais utilizada, por oferecer melhor desempenho e compatibilidade com todas as placas.


C/C++ (Arduino IDE):


O Arduino é programado principalmente em C/C++, usando a Arduino IDE, que adapta essas linguagens para facilitar o uso. Você usa funções como:

  • void setup() { }
  • void loop() { }


Python:


O Python pode ser usado em alguns modelos de Arduino compatíveis com:

  • MicroPython
  • CircuitPython


Mas nem todos os Arduinos suportam Python. Geralmente é usado em placas mais novas, como:

  • Arduino Nano 33 BLE.
  • Arduino Portenta
  • ESP32 (não é Arduino, mas usa MicroPython)
