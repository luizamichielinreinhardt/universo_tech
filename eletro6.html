O que são entradas e saídas digitais em circuitos integrados como o Arduíno







Entradas e Saídas digitais:

As entradas e saídas digitais são pinos usados pelos circuitos, como o Arduino, para se comunicar com o mundo externo. As entradas recebem sinais que podem estar ligados (1) ou desligados (0), como quando um botão é pressionado ou um sensor detecta algo. Já as saídas enviam esses mesmos sinais para controlar dispositivos, permitindo ligar LEDs, buzzer ou acionar motores. Assim, entradas percebem o ambiente e saídas fazem o Arduino agir sobre ele.






O que é uma entrada digital?


Uma entrada digital é um pino usado para receber informações do ambiente. Ela só entende dois valores:

  • HIGH (1) → tensão alta (geralmente 5V ou 3.3V)
  • LOW (0) → tensão baixa (0V)


Exemplos de sensores usados como entradas digitais:

  • Botões.
  • Chaves (switches).

No Arduino:

  • pinMode(2, INPUT); // Configura o pino 2 como entrada int valor = digitalRead(2); // Lê 1 ou 0


O que é uma saída digital?


Uma saída digital é um pino que envia sinais para controlar dispositivos. Assim como a entrada, ela só envia dois estados:

  • HIGH (1) → liga o dispositivo
  • LOW (0) → desliga o dispositivo

Exemplos de dispositivos controlados por saídas digitais:

  • LEDs.
  • Relés.

No Arduino:

  • pinMode(13, OUTPUT); // Pino 13 como saída digitalWrite(13, HIGH); // Liga digitalWrite(13, LOW); // Desliga
