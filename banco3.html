O que é um dicionário de dados? Quais os tipos de dados?







Um dicionário de dados é um documento ou ferramenta que descreve todos os elementos de um banco: tabelas, campos, tipos, tamanhos, relacionamentos e regras. Ele serve para padronizar o desenvolvimento, evitar erros e facilitar o entendimento do banco.


INT – números inteiros:


  • O tipo de dado INT é usado para armazenar números inteiros, ou seja, valores sem casas decimais. Ele é muito utilizado em contagens, identificadores (como IDs de usuários) e qualquer informação que envolva números exatos. Por ser um tipo simples e eficiente, o INT é um dos formatos mais usados dentro de tabelas de banco de dados.

FLOAT / DOUBLE – números decimais:


  • Os tipos FLOAT e DOUBLE armazenam números que possuem casas decimais. Eles são usados quando é necessário trabalhar com valores que não são inteiros, como notas escolares, medidas físicas, valores científicos ou cálculos matemáticos. O DOUBLE tem maior precisão do que o FLOAT, sendo preferido em situações que exigem cálculos mais exatos.

CHAR / VARCHAR – textos curtos:


  • Os tipos CHAR e VARCHAR servem para armazenar textos curtos. O CHAR tem tamanho fixo, sendo usado para dados que sempre têm o mesmo número de caracteres, como códigos ou siglas. Já o VARCHAR tem tamanho variável e é ideal para textos como nomes, e-mails e títulos. Eles são muito usados porque economizam espaço e permitem organizar dados de forma eficiente.

TEXT – textos grandes:


  • O tipo TEXT é destinado ao armazenamento de textos longos, como descrições detalhadas, comentários, artigos, postagens de blog ou campos de observação. Diferente do VARCHAR, ele suporta grandes quantidades de caracteres. É ideal quando o conteúdo não tem um tamanho previsível e pode variar muito entre uma linha e outra.

DATE, TIME, DATETIME – datas e horários:


  • Os tipos DATE, TIME e DATETIME são usados para guardar informações relacionadas a datas e horários. O DATE armazena apenas o dia, mês e ano; o TIME registra somente horas, minutos e segundos; e o DATETIME combina ambos, sendo muito utilizado para registrar momentos exatos, como a data de criação de um usuário ou o horário de uma venda. Eles são essenciais em sistemas que precisam acompanhar eventos ao longo do tempo.

BOOLEAN – verdadeiro/falso:


  • O tipo BOOLEAN armazena valores lógicos, sendo representado como true ou false. Ele é utilizado para indicar condições simples, como "ativo ou inativo", "sim ou não", "ligado ou desligado". Apesar de simples, o Boolean facilita muito a tomada de decisões dentro do sistema e ajuda a organizar dados que dependem de uma condição.
