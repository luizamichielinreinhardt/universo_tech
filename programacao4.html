O que é o sistema de arquivos e como



funciona no CMD e no Explorer?





Sistema de Arquivos:

O sistema de arquivos é a forma como o computador organiza, armazena e localiza os arquivos dentro de um dispositivo (HD, SSD, pen drive etc.). Ele funciona como uma “biblioteca”, definindo como os arquivos são nomeados, divididos, salvos e encontrados dentro das pastas.


Modo Texto (CMD)


Como funciona: No modo texto, como no CMD, você controla o sistema de arquivos usando comandos digitados. Nesse modo, você não vê ícones nem janelas — apenas texto.


Exemplos de ações no CMD:

  • Listar arquivos: dir
  • Entrar em uma pasta: cd nome_da_pasta
  • Criar pasta: mkdir nome
  • Apagar arquivos: del arquivo.txt

Explorer


Como funciona: um navegador de arquivos visual do Windows. Ele permite que o usuário veja pastas e documentos por meio de ícones e janelas, tornando a navegação simples e intuitiva.


Exemplos de ações no Explorer:

  • Abrir pastas clicando duas vezes.
  • Arrastar arquivos para mover ou copiar.
  • Renomear usando o botão direito.
  • Ver arquivos com visualização em lista, ícones ou detalhes.
