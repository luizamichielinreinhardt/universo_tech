O que é uma linguagem de programação interpretada?


Uma linguagem de programação interpretada é aquela cujo código é lido e executado linha por linha por um interpretador. Ela não precisa gerar um arquivo executável antes de rodar, o que a torna mais fácil de testar e modificar rapidamente. Geralmente é mais flexível, embora possa ser menos rápida do que uma linguagem compilada. Entre os exemplos mais conhecidos estão Python, JavaScript e PHP. Nesse tipo de linguagem, o funcionamento é simples: você escreve o código, o interpretador lê e executa imediatamente.


O que é uma linguagem de programação compilada?


Já uma linguagem de programação compilada precisa passar antes por um processo chamado compilação. Nesse processo, um programa chamado compilador converte o código-fonte em código de máquina, gerando um arquivo executável. Esse tipo de linguagem costuma produzir programas mais rápidos e é ideal para sistemas complexos, jogos e aplicações pesadas. Durante a compilação, o código também passa por análises, otimizações e verificações antes de ser executado. Entre os principais exemplos estão C, C++ e Rust. O funcionamento ocorre assim: você escreve o código, o compilador traduz tudo, gera o executável e então o computador o executa.
