O que é uma modelagem de banco de dados? E o modelo DER? MER?







Modelagem de banco de dados é o processo de planejar como os dados serão organizados antes da criação do banco. Ela define tabelas, relacionamentos, regras e restrições.


DER (Diagrama Entidade-Relacionamento):


  • Representa entidades (tabelas), seus atributos (colunas) e relacionamentos (1 para 1, 1 para N, N para N). É usado na fase de modelagem conceitual.

MER (Modelo Entidade-Relacionamento):


  • É o modelo teórico que define entidades, atributos e relacionamentos. O DER é o desenho/diagrama visual que representa o MER.
