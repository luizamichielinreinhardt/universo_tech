O que é HTML e CSS?








HTML (HyperText Markup Language):


é a linguagem usada para criar a estrutura de uma página. Ela define os elementos que vão aparecer no site, como títulos, parágrafos, imagens, links, tabelas, botões e outros conteúdos. Em outras palavras, o HTML organiza o que aparece na tela e a forma como os elementos estão distribuídos, funcionando como o “esqueleto” de um site.


CSS (Cascading Style Sheets):


é a linguagem responsável pela aparência visual da página. Ela define cores, tamanhos, fontes, margens, alinhamentos, espaçamentos, animações e todo o estilo que deixa o site bonito e organizado. Se o HTML é o esqueleto, o CSS é como a “roupa” e o “design” que dão forma e estilo ao site.
